In this project I learned how to display images in different projections

How I displayed the map in different projections using QGIS

In order to display different map projections, I began with adding a vector layer of a world map. Through changing the EPSG in QGIS, I was able to display the different map projections. The indicatrix mapper plug-in was activated to then show the circles across the maps showing the distortion of area, size, and shape.
