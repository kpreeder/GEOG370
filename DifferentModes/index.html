Different Modes Categorizing Average Income Per County

Equal Count

The Equal Count Distribution creates intervals so that each has an equal proportion of the sample. Pros: Each class will be equally represented on the map. Cons: Some counts may be inequal if sample sizes are uneven, may group outliers with data and give pooer representations.

Equal Interval

The Range of values are divided into intervals of the same size. Pros: Good for showing values that are either over or under represented Cons: Intervals that are over represetned will result in choropleht maps that appear mostly one color, as you can see here especially.

Natural Breaks

Places values into naturally occuring data categories. Pros: Minimizes within class variances and maximizes variance between groups Cons: Not good with data sets with low variance

Using the total individual income in each county, and dividing that by the population in each county, I found my averaged data. The reason for using this ratio is that it better represents each county because it takes into account the population differences that would skew the data.
